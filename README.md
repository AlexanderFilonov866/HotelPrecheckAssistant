# Hotel Precheck Assistant

Internal-style HTML tool for hotel reservation precheck specialists.

A dynamic checklist and call-script assistant designed to help accommodation verification agents conduct structured booking checks without missing critical questions.

## Features

- Adaptive call verification scripts  
- Property-type specific scenarios:
  - Hotels
  - Apart-hotels
  - Apartments
  - Guest houses
  - Hostels

- Dynamic rule engine:
  - Star-rating based logic
  - Country-specific compliance questions
  - Risk and warning flags
  - Conditional question flows

- Bilingual interface
  - Russian
  - English

- Built-in guest comment templates

- Additional verification checks:
  - Taxes and extra fees
  - Child policies
  - Bedding confirmation
  - Check-in / check-out conditions
  - Pets policy

- Responsive single-file offline HTML tool
- No dependencies
- Can be used as an internal productivity assistant or training support tool

## Purpose

The project is designed to improve consistency, reduce missed checks, and help precheck specialists navigate complex booking verification scenarios more confidently.

## Tech
- HTML
- CSS
- Vanilla JavaScript
